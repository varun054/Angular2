Steps to create your first Angular2 app.

Step 1: Install node and npm in your system using below commands.
sudo apt-get update
sudo apt-get install nodejs
sudo apt-get install npm

Step 2: Install type script in your system using below command.
npm install -g typescript

Step 3: Create your Project Folder using below command. 
mkdir HelloWorld

Step 4: Enter in project directory using below command.
cd HelloWorld

Step 5: create package.json, tsconfig.json, typings.json, systemjs.config.js files in project root folder. copy the same file code from below URL.
https://angular.io/docs/ts/latest/quickstart.html
and paste them your respective files which you created in your root folder.

Step 6: Now load the node packages using below command.
npm install

Step 7: Create a new directory app and enter inside the directory.
mkdir app
cd app

Step 8: create a new file app.component.ts and paste the same file code in HelloWorld/app/app.component.ts file.

Step 9: create a new file app.module.ts and paste the same file code in HelloWorld/app/app.module.ts file.

Step 10: create a new file main.ts and paste the same file code in HelloWorld/app/main.ts file.

Step 11: Now you have done everything and Run the application.go in your project root folder and run the application.
npm start
